### 👨‍💻 Sobre mim

Desenvolvedor Full Stack focado na construção de sistemas **robustos e escaláveis**, com forte compromisso com a **qualidade de código** e a **segurança da informação**. Minha abordagem de desenvolvimento é pautada por rigor técnico, boas práticas e evolução constante.

- 🛠️ **Stack principal:** Python, NestJS, Next.js, TypeScript, Node.js
- 🗄️ **Banco de dados:** PostgreSQL, Prisma ORM
- ☁️ **Cloud:** AWS
- 🎯 **Foco:** APIs RESTful, Arquitetura escalável, Segurança
- 🧠 **Filosofia:** Código limpo hoje = menos dor amanhã
